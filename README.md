## 📂 Repository Structure & Projects

This workspace is divided into three standalone engineering applications. **Please click into each project folder for a detailed mathematical breakdown, source code, and instructions.**

### [🎛️ 1. Digital Audio Echo Generator](./Project_1_Echo_Generator)
* **Domain:** 1D Digital Signal Processing (DSP)
* **Summary:** An automated algorithm that manipulates discrete 1D arrays to superimpose a time-delayed, normalized echo effect onto audio signals. Features continuous-to-discrete signal mapping and dynamic amplitude scaling to prevent audio clipping.

### [📈 2. Matrix Saddle Point Finder](./Project_2_Saddle_Points)
* **Domain:** 2D Matrix Optimization & 3D Visualization
* **Summary:** A computational script that evaluates mathematical matrices to identify "saddle points" (minimax points) using nested iterative logic. Includes a visualization module that generates topographical 3D surface plots of the evaluated data.

### [🖼️ 3. Digital Image Blurring (Spatial Filter)](./Project_3_Image_Blur)
* **Domain:** Computer Vision & Image Processing
* **Summary:** A custom-built spatial mean filter that applies a dynamic blurring effect to grayscale images. The algorithm dynamically calculates moving averages across a user-defined matrix window while cleanly handling boundary pixels without inducing edge artifacts.
